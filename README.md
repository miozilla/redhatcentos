# redhatcentos 🪙🌶️🎩
redhatcentos : Cloud SDK Virtual Machine | Red Hat Enterprise Level (RHEL) 9, CentOS 9, RPM, yum |

## Objectives 
- To install Cloud SDK to a virtual machine.
- Initialize & run core gcloud commands from command-line.
- Cloud SDK RPM packages are supported for Red Hat Enterprise Level (RHEL) 9 & CentOS 9.
